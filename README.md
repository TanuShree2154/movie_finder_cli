# Movie Finder CLI

A Python CLI application that fetches movie details using the OMDb API and stores search history using CSV file handling.

---

## Features

* Search movies using OMDb API
* Display movie details in terminal
* Handle invalid movie searches
* Store search history in CSV file
* View previous searches
* Exception handling using try-except
* Menu-driven CLI interface

---

## Technologies Used

* Python
* Requests Library
* CSV File Handling
* OMDb API

---

## Concepts Practiced

* API Integration
* JSON Handling
* File Handling
* Exception Handling
* Functions
* Loops and Conditions
* CLI Application Development

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/movie-finder-cli.git
```

2. Navigate to the project folder:

```bash
cd movie-finder-cli
```

3. Install dependencies:

```bash
pip install requests
```

---

## How to Run

```bash
python movie_finder.py
```
